# 🧪 دليل الاختبار الشامل - نظام كشف وحظر الأجهزة المتعددة

## 📋 المتطلبات

- ✅ هاتفان أندرويد مختلفان
- ✅ حساب مستخدم واحد للاختبار
- ✅ تطبيق الأدمن مثبت ومفتوح
- ✅ اتصال إنترنت نشط

---

## 🧪 سيناريو 1: مستخدم جديد - جهاز واحد

### الخطوات:

1. **على الهاتف الأول:**
   ```
   ✓ افتح التطبيق
   ✓ سجل حساب جديد أو سجل دخول
   ```

2. **في Firebase Console:**
   ```
   ✓ افتح Firestore
   ✓ اذهب إلى users → [userId]
   ✓ تحقق من:
     - deviceUsed = 1
     - useMultipleDevice = false
     - activeDeviceId = [device_id]
     - deviceList يحتوي على جهاز واحد
   ```

3. **في تطبيق الأدمن:**
   ```
   ✓ ابحث عن المستخدم
   ✓ يجب أن تشاهد:
     - "يستخدم جهاز واحد ✓"
     - deviceUsed: 1
     - لون أخضر أو عادي (بدون تحذير)
   ```

### النتيجة المتوقعة:

- ✅ المستخدم يدخل بنجاح
- ✅ لا توجد رسائل حظر
- ✅ تطبيق الأدمن يعرض "جهاز واحد"

## 🔴 سيناريو 2: نفس المستخدم - جهاز ثاني (محاولة نصب)

### الخطوات:

1. **على الهاتف الثاني:**
   ```
   ✓ افتح التطبيق
   ✓ سجل دخول بنفس الحساب المستخدم في الهاتف الأول
   ```

2. **ماذا يجب أن يحدث فوراً:**
   ```
   ✅ الشاشة: رسالة حظر تظهر فوراً
   ✅ العنوان: "🚫 تم حظر حسابك"
   ✅ الرسالة: "تم حظر حسابك بسبب استخدام أجهزة متعددة.
               
               يُسمح فقط باستخدام جهاز واحد لكل حساب.
               
               للاستفسار أو فك الحظر، يرجى التواصل مع الدعم الفني."
   ✅ الزر: "حسناً"
   ```

3. **بعد الضغط على "حسناً":**
   ```
   ✅ تسجيل خروج تلقائي
   ✅ العودة لشاشة تسجيل الدخول
   ✅ لا يستطيع الدخول مرة أخرى
   ```

4. **في Firebase Console:**
   ```
   ✓ افتح Firestore
   ✓ اذهب إلى users → [userId]
   ✓ تحقق من:
     - deviceUsed = 2 ✅
     - useMultipleDevice = true ✅
     - isDeviceLocked = true ✅
     - activeDeviceId = [device_2_id]
     - deviceList يحتوي على جهازين
   ```

5. **في تطبيق الأدمن:**
   ```
   ✓ ابحث عن المستخدم
   ✓ يجب أن تشاهد:
     - "⚠️ يستخدم أجهزة متعددة"
     - deviceUsed: 2
     - لون أحمر أو برتقالي (تحذير)
     - قائمة الأجهزة تعرض جهازين
     - زر "فك حظر المستخدم" نشط
   ```

### النتيجة المتوقعة:

- ❌ المستخدم لا يستطيع الدخول من الهاتف الثاني
- ✅ رسالة حظر واضحة
- ✅ تطبيق الأدمن يعرض "أجهزة متعددة"
- ✅ العداد = 2

## 🔓 سيناريو 3: فك الحظر من الأدمن

### الخطوات:

1. **في تطبيق الأدمن:**
   ```
   ✓ ابحث عن المستخدم المحظور
   ✓ اضغط على "فك حظر المستخدم"
   ✓ أكد العملية
   ```

2. **في Firebase Console:**
   ```
   ✓ تحقق من:
     - isDeviceLocked = false ✅
     - blockUser = false ✅
     - deviceUsed = 1 (يتم إعادة تعيينه)
     - useMultipleDevice = false
   ```

3. **على الهاتف الأول (الأصلي):**
   ```
   ✓ افتح التطبيق
   ✓ سجل دخول
   ✓ يجب أن يدخل بنجاح ✅
   ```

4. **على الهاتف الثاني:**
   ```
   ✓ حاول تسجيل الدخول
   ✓ يجب أن ترى رسالة حظر مرة أخرى ❌
   ```

### النتيجة المتوقعة:

- ✅ المستخدم يستطيع الدخول من الهاتف الأصلي فقط
- ❌ لا يستطيع الدخول من الهاتف الثاني
- ✅ تطبيق الأدمن يعرض "جهاز واحد" مرة أخرى

## 🔨 سيناريو 4: حظر يدوي من الأدمن

### الخطوات:

1. **في تطبيق الأدمن:**
   ```
   ✓ ابحث عن أي مستخدم (حتى لو يستخدم جهاز واحد)
   ✓ اضغط على "حظر المستخدم"
   ✓ اكتب سبب الحظر: "انتهاك شروط الاستخدام"
   ✓ أكد العملية
   ```

2. **في Firebase Console:**
   ```
   ✓ تحقق من:
     - blockUser = true ✅
     - blockReason = "انتهاك شروط الاستخدام"
   ```

3. **على هاتف المستخدم (إذا كان مفتوح):**
   ```
   ✅ خلال ثواني، يجب أن تظهر رسالة:
     - العنوان: "🚫 تم حظر حسابك"
     - الرسالة: "انتهاك شروط الاستخدام"
     - تسجيل خروج تلقائي
   ```

4. **إذا حاول تسجيل الدخول مرة أخرى:**
   ```
   ✓ يكتب البريد والرقم السري
   ✓ يضغط "تسجيل دخول"
   ✅ رسالة حظر تظهر فوراً
   ✅ لا يستطيع الدخول
   ```

### النتيجة المتوقعة:

- ❌ المستخدم محظور تماماً
- ✅ رسالة حظر مخصصة
- ✅ كشف فوري (Realtime)

## 🐛 الأخطاء المحتملة وحلولها

### المشكلة 1: العداد لا يتحدث

**السبب:**

- `registerDevice()` لا يُستدعى في `LoginActivity`

**الحل:**

- ✅ تم إصلاحه في التحديث الأخير

### المشكلة 2: المستخدم يدخل رغم الحظر

**السبب:**

- لا يتم التحقق من `isDeviceLocked`

**الحل:**

- ✅ `DeviceManager` يتحقق من كلا الحقلين الآن

### المشكلة 3: رسالة الحظر لا تظهر فوراً

**السبب:**

- Realtime Listener غير مفعل

**الحل:**

- ✅ `setupBlockStatusListener()` في `MainActivity`

### المشكلة 4: تطبيق الأدمن يعرض بيانات قديمة

**السبب:**

- Firebase Console قد يكون لديه تأخير في العرض

**الحل:**

- انتظر 5-10 ثواني
- اضغط Refresh في Firebase Console
- أعد فتح تطبيق الأدمن
