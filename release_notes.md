# نصوص - ملاحظات الإصدار

## الإصدار 1.1.0 (versionCode: 2)

### ميزات جديدة ✨
- مكتبة رقمية شاملة للكتب العربية والإنجليزية
- قارئ PDF متقدم مع إمكانية البحث والإشارات المرجعية
- دعم كامل للغة العربية مع تخطيط RTL
- نظام مصادقة آمن مع Google Sign-in
- سلة تسوق متطورة مع دفع آمن
- قراءة بدون اتصال بالإنترنت

### الأمان والخصوصية 🔒
- تشفير AES-256-GCM للبيانات المحلية
- حماية الشبكة مع TLS 1.3 وCertificate Pinning
- متوافق مع قوانين GDPR الأوروبية
- إدارة ذكية للموافقات والخصوصية
- تشفير end-to-end للبيانات الحساسة

### تحسينات التطبيق 🚀
- واجهة مستخدم محسنة ومتجاوبة
- أداء محسن مع ProGuard optimization
- دعم متعدد اللغات (العربية والإنجليزية)
- إشعارات ذكية لتحديثات المكتبة
- مزامنة سحابية آمنة مع Firebase

### الميزات التقنية 🔧
- Android 14 (API 35) مدعوم
- متوافق مع الأجهزة من Android 7.0+
- حجم التطبيق محسن (~12MB)
- دعم 64-bit architecture
- Battery optimization مُحسن
